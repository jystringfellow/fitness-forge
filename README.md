# fitness-forge

Forge your daily workout.

## Starter app (Expo + React Native + TypeScript)

This repo now includes a V1 starter built with Expo Router for iOS, Android, and web.

### Included features

- Quick Generate screen with selectors for time, energy, focus, and attachment
- Workout result screen with cardio block + single attachment block
- Exercise library
- Favorites (local storage)
- History (local storage)
- Settings placeholder for phase 2 ideas

### Run locally (pnpm)

```bash
pnpm install
pnpm start
```

Then open on iOS, Android, or web through Expo.

### Useful commands

```bash
pnpm typecheck
pnpm web
pnpm ios
pnpm android
```
